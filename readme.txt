This plugin allows you to display an inventory manager for rpgs in your posts using the shortcode [inventory][/inventory].

== Description ==

You can use this plugin to show an inventory manager.

For example:

`
[rp-inventory name='MyHero'][/rp-inventory]
`

== Installation ==

1. Upload <strong>sonnenstrasse-inventory</strong> folder to the <strong>/wp-content/plugins/</strong> directory.
2. Activate the plugin through the <strong>Plugins</strong> menu in WordPress.
3. That's all.
